# IMS App Mobile (Expo 54)

This is a fresh Expo SDK 54 scaffold with a Login and Home screen.

Install and start (PowerShell):

```powershell
cd "d:\project\New folder\ImsApp\mobile-expo54"
npm install
npm start
```

Notes:
- Default BACKEND_BASE is set to `http://10.0.2.2:5000` in `app.json` extras. Update it if needed.
- For Android emulator use `10.0.2.2`; for iOS simulator use `localhost`; for physical devices use your PC's LAN IP.
